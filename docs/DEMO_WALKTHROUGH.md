# Демонстрация функционала (локально)

## Подготовка

1. Установите зависимости и примените миграции: `pip install -r requirements.txt`, `python manage.py migrate`.
2. Создайте суперпользователя: `python manage.py createsuperuser` (для Django Admin).
3. В `.env` укажите `BOT_TOKEN`, `BACKEND_API_URL=http://127.0.0.1:8000`.
4. Узнайте свой **числовой Telegram ID** (через @userinfobot или аналог) и добавьте в `.env` строку  
   `ORGANIZER_BOOTSTRAP_TELEGRAM_IDS=ВАШ_ID` — так вы получите право **создавать хакатоны** до любых назначений в админке.
5. Запустите backend: `python manage.py runserver`.

## Mini App

**Через Django (удобно):** откройте в браузере `http://127.0.0.1:8000/miniapp/`.

**С файла на диске:** откройте `mini_app/index.html` в браузере с параметрами  
`?api_base=http://127.0.0.1:8000&telegram_id=ВАШ_ID`.

### Сценарий участника / капитана

1. Зарегистрируйтесь в боте (`/start`, ФИО, email, навыки, роль).
2. В Mini App: **Профиль** — данные; при роли капитана — форма **создания команды**.
3. **Поиск команды** — список команд, отклик.
4. **Панель капитана** (если вы капитан с командой) — настройки, заявки, блок **Хакатоны** — подключение команды к событию.

### Сценарий организатора

1. Убедитесь, что `ORGANIZER_BOOTSTRAP_TELEGRAM_IDS` содержит ваш ID **или** в Admin → Hackathon вас добавили в «организаторы».
2. Обновите главный экран Mini App: должна появиться кнопка **«Создать хакатон»**.
3. Заполните форму и отправьте — в ответе id/slug созданного события; вы автоматически в списке организаторов.

## Бот

1. Запустите бота (из корня проекта): `python -m bot.main` (или как у вас принято).
2. `/start` — после регистрации в клавиатуре появятся:
   - **➕ Новый хакатон** — если `can_create_hackathon` (bootstrap или уже организатор);
   - **📊 Выгрузки организатора** — если вы организатор хотя бы одного хакатона.
3. Пройдите мастер создания хакатона или выгрузите Excel (участники / команды).

## Альтернатива: только Django Admin

- Создайте `Hackathon`, укажите `slug`, ссылку на таблицу, отметьте **организаторов** (User с нужным `telegram_id` должен существовать — проще сначала зарегистрироваться в боте).
- Связи «команда ↔ хакатон» капитан добавляет через Mini App или вы вручную через модель `HackathonTeam`.

## Полезные API (Swagger)

С `/api/docs/` при запущенном сервере можно вызывать в том числе:

- `GET /api/hackathons/permissions/?telegram_id=`
- `POST /api/hackathons/create/`
- `GET /api/hackathons/?captain_telegram_id=`
- `POST /api/hackathons/<id>/join-team/`
